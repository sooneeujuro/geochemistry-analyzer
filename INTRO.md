# 지구화학 데이터 분석기

지구화학 데이터를 쉽고 빠르게 분석하고 시각화하는 웹 기반 도구입니다.

---

## 🎯 이런 분들에게 추천해요

- 📊 지구화학 데이터를 시각화하고 싶은 연구자
- 🔬 상관관계 분석이 필요한 학생
- 📈 논문용 고품질 그래프가 필요한 분
- 💻 코딩 없이 데이터 분석을 하고 싶은 분

---

## 🚀 빠른 시작

### 1단계: 파일 업로드
- Excel (`.xlsx`, `.xls`) 또는 CSV 파일을 준비하세요
- 첫 번째 행은 컬럼 이름이어야 합니다
- 드래그 앤 드롭 또는 클릭하여 업로드

### 2단계: 변수 선택
- X축과 Y축에 분석할 변수를 선택하세요
- **단일 컬럼**: 하나의 수치형 컬럼 (예: SiO₂, Al₂O₃)
- **비율**: 두 컬럼의 비율 계산 (예: SiO₂/TiO₂)

### 3단계: 분석 및 시각화
- 자동으로 산점도가 생성됩니다
- 상관계수, 회귀선 등 통계 결과를 확인하세요
- 차트 스타일, 색상, 오차범위 등을 자유롭게 조정하세요

### 4단계: 내보내기
- **PNG 저장**: 논문용 고해상도 이미지
- **SVG 저장**: 벡터 그래픽 (편집 가능)

---

## ✨ 주요 기능

### 📁 데이터 관리
- Excel/CSV 파일 지원
- 타입별 색상 구분 (암석 타입, 샘플 그룹 등)
- 비율 계산 자동화

### 📊 통계 분석
- 피어슨/스피어만 상관계수
- 선형 회귀 분석 (R², p-value)
- 타입별 독립적인 통계

### 📈 고급 시각화
- X/Y축 로그 스케일
- 1:1 비율 유지 및 참조선
- 축 반전 (깊이 프로파일용)
- 오차범위 표시 (5가지 방법)

### 🎨 차트 커스터마이징
- 과학적 표기법 (10⁴ 위첨자)
- 마커 모양/크기/색상 조정
- 추세선 스타일 설정
- 데이터 라벨 표시

### 🔍 스캔 모드
- 모든 변수 조합 자동 분석
- AI 추천 기능 (유의미한 패턴 발견)
- PCA 분석

## 💡 사용 팁

### 타입별 분류 활용
데이터에 암석 타입, 샘플 그룹 등의 분류 컬럼이 있다면:
1. "타입 분류 설정"에서 해당 컬럼 선택
2. 각 타입이 다른 색상으로 표시됩니다
3. 타입별 독립적인 통계 분석 가능

### 오차범위 표시
측정 오차나 불확실성을 표현하려면:
1. **백분율**: 측정 오차가 일정한 경우 (예: ±5%)
2. **고정값**: 모든 데이터의 오차가 동일한 경우
3. **컬럼에서 읽기**: 엑셀에 오차값이 별도 컬럼으로 있는 경우
4. **표준편차/표준오차**: 통계적 불확실성 표현

### 논문용 그래프
고품질 그래프가 필요하다면:
1. 폰트 크기 조정 (축 제목 14-16px, 숫자 12px)
2. 마커 크기 적절히 조정 (60-80)
3. 배경색 흰색 유지
4. **SVG로 저장** → Illustrator/Inkscape에서 편집

## 🎓 예시 시나리오

### 시나리오 1: 기본 상관관계 분석
```
데이터: 화강암 지구화학 분석 결과
목표: SiO₂와 K₂O의 상관관계 확인

1. 엑셀 파일 업로드
2. X축: SiO₂, Y축: K₂O 선택
3. 타입 컬럼: Rock_Type 선택 (화강암, 화강섬록암 등)
4. 상관계수 및 회귀선 확인
5. PNG로 저장
```

### 시나리오 2: 희토류 원소 패턴
```
데이터: 희토류 원소 농도
목표: La/Yb 비율 분석

1. 파일 업로드
2. X축: Sample_Number (단일), Y축: La/Yb (비율)
3. 로그 스케일 적용 (Y축)
4. 표준편차로 오차범위 표시
5. 과학적 표기법 (위첨자) 사용
```

### 시나리오 3: 전체 데이터 탐색
```
데이터: 다양한 지구화학 원소 (20개 컬럼)
목표: 유의미한 상관관계 발견

1. 파일 업로드
2. "스캔 모드" 선택
3. AI 추천 켜기
4. 유의한 조합 자동 발견
5. 관심 조합 클릭 → 상세 분석
```

## 🆘 자주 묻는 질문

### Q: 데이터가 안전한가요?
A: 모든 처리는 브라우저에서 이루어지며, 서버에 데이터가 전송되지 않습니다. (선택적으로 Supabase에 저장 가능)

### Q: 최대 데이터 크기는?
A: 브라우저 메모리에 따라 다르지만, 일반적으로 수만 개의 데이터 포인트까지 원활하게 작동합니다.

### Q: 모바일에서 사용 가능한가요?
A: 반응형 디자인으로 태블릿에서는 사용 가능하나, PC 환경을 권장합니다.

### Q: 그래프를 편집할 수 있나요?
A: SVG 형식으로 저장하면 Adobe Illustrator, Inkscape 등에서 자유롭게 편집 가능합니다.
