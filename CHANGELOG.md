## [0.5.6] - 2024-11-24

### ✨ Added (새 기능)
- **로그 스케일**: X축/Y축 독립적인 로그 스케일 지원
- **축별 숫자 형식**: X축과 Y축을 각각 다른 형식으로 표시 (일반/과학적/천단위)
- **위첨자 지수 표기**: 10⁴ 형식의 과학적 표기법
- **1:1 비율 유지**: 정사각형 플롯 생성 옵션
- **1:1 참조선**: y=x 대각선 표시
- **눈금 간격 조정**: X/Y축 눈금 간격 수동 설정
- **차트 제목**: 사용자 정의 제목 추가 기능
- **축 제목 위치 조정**: 숫자와 겹치지 않도록 위치 미세 조정
- **축 반전**: X축/Y축 반전 (지질학 깊이 프로파일용)
- **SVG 내보내기**: 벡터 그래픽 형식으로 저장
- **데이터 라벨**: 각 포인트에 타입 이름 표시
- **사용자 정의 색상**: 타입별 색상 팔레트 커스터마이징 (8색)

### 🎨 Changed (변경)
- **반응형 레이아웃**: 큰 화면에서 옵션과 차트를 좌우 배치
- **패널 하이라이트**: 열린 패널을 색상으로 구분 (파란색/보라색/초록색)
- **PNG 저장 버튼**: "이미지 저장" → "PNG 저장"으로 명칭 변경

### 🐛 Fixed (버그 수정)
- 타입 컬럼 선택 시 색상이 적용되지 않는 버그 수정
- 로그 스케일 사용 시 domain 타입 오류 수정
- 눈금 간격 설정 시 무한 루프 방지
